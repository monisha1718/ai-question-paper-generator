```
└── .gitignore            # Files ignored by Git
```

## Installation

Clone the repository

```
git clone https://github.com/monisha1718/ai-question-paper-generator.git
```

Go to the project folder

```
cd ai-question-paper-generator
```

Install required packages

```
pip install -r requirements.txt
```

Run the application

```
streamlit run app.py
```

## Usage

1. Enter the subject or topic  
2. Select the difficulty level  
3. Click **Generate Question Paper**  
4. The AI will generate questions automatically
